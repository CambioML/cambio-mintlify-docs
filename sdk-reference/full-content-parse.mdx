## Overview
Using `AnyParser`, you can parse the full content from your documents into markdown.

## Setup
Refer to the [Quickstart guide](/quickstart/#setup) to install the AnyParser SDK and get your api key.

Next, set up your `AnyParser` sync or async client.

```python anyparser_sync.py
from any_parser import AnyParser

ap = AnyParser(api_key="...")

md_output, total_time = ap.extract(file_path="./data/test.pdf")
```

```python anyparser_async.py
from any_parser import AnyParser

ap = AnyParser(api_key="...")

file_id = ap.async_extract(file_path="./data/test.pdf")

md = ap.async_fetch(file_id=file_id)
```

## Choose a Model (`base` or `pro`)

The `base` model is a general-purpose model that is optimized for a wide range of document types. The `pro` model is a more advanced model that is optimized for more complex documents.


### BASE Model (default) Example

To use the `BASE` model, simply import the `ModelType` and pass the `model` parameter as shown below.
```python
from any_parser import ModelType

# start the extraction request
file_id = ap.async_extract(example_local_file,  model=ModelType.BASE)
# fetch results (5s polling up to 60s)
markdown_string = ap.async_fetch(file_id, sync=True, sync_timeout=60, sync_interval=5)
```

### PRO Model Example
To use the `PRO` model, simply import the `ModelType` and pass the `model` parameter as shown below.
```python
from any_parser import ModelType

# start the extraction request
file_id = ap.async_extract(example_local_file,  model=ModelType.PRO)
# fetch results (5s polling up to 60s)
markdown_string = ap.async_fetch(file_id, sync=True, sync_timeout=60, sync_interval=5)
```
